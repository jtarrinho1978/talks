Getting started

# Installing BBB
[sudo] npm install grunt-bbb

# Create a new project
bbb init

Module scaffolding

# Create a new module
bbb init:module

Building release assets

# Precompile all templates and concatenate them with all JavaScript
bbb debug
# Minify
bbb release

Running the development server

# Development
bbb server
# Build test
bbb server:debug
# Production test
bbb server:release

Workflow philosophy

  • Never re-build in development
  • Never touch index.html
  • Deployment remaps
  • Build all source files into as few files as possible
